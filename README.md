## Products Overview

### Tailor Resume MCP

An enterprise-grade, AI-powered resume customization platform that solves critical problems in HR technology:
- **Market**: HR Technology / ATS / Resume Optimization ($2.5B+ market)
- **Key Innovation**: First resume platform built on MCP architecture with 100% parsing accuracy
- **Impact**: Can help millions of job seekers and save enterprises $50K-$500K annually

### TraceBuddy

A React Native iOS app for coordinating group travel with real-time location tracking:
- **Market**: Travel Technology / Location Services ($1.2T travel app market)
- **Key Innovation**: Real-time group coordination with seamless Apple Maps integration
- **Impact**: Can improve safety and coordination for millions of group travelers
